# 国有企业合同审查技能（央企/国企版）

本技能是一套专为中央企业、地方国有企业及其各级子企业设计的**合同审查辅助工具**，在通用合同法律风险审查基础上，深度融合中国大陆国资监管框架下的合规审查维度，帮助法务、合规、风控人员系统识别合同文本风险与缔约程序风险。

## ✨ 功能特性

- **双层审查框架**：同时覆盖合同条款本身的法律风险与缔约过程/决策程序的合规性。
- **国企专属核查维度**：内置招标采购合规、三重一大决策、国有资产交易审批、关联交易、廉洁风控、境外投资合规等 9 大专项核查。
- **分类审查清单**：针对采购/物资、工程施工/EPC、政府采购、股权转让、关联交易、境外投资等常见合同类型提供差异化审查要点。
- **风险等级预警**：结合市场/行业惯例基准，自动给出黄色/红色预警信号。
- **结构化输出**：生成包含执行摘要、关键条款风险分析、国资合规专项核查表、缺失条款提示的完整报告。

## 🎯 适用场景

- 国企法务、合规、风控部门审查对外签署的各类合同；
- 审计、巡视、纪检人员核查合同合规性；
- 外部律师为国企提供合同审查服务时的参考框架；
- 国企内部合同管理系统的规则引擎基础。

## 🚀 快速开始

1. 将 `skill.md` 导入您的 AI 助手（如支持自定义技能的平台）。
2. 上传待审查的合同文本，并告知企业属性、合同类型、金额等必要背景。
3. 按技能提示获取结构化审查报告。
4. 根据报告中的“国资合规专项核查”逐项落实整改或补充材料。

## 📝 使用示例

参见 `examples/sample_report.md`，展示了一份 EPC 总承包合同的完整审查报告。

## ⚠️ 注意事项

- 本技能仅为风险识别参考，**不能替代**企业内部合规审批、总法律顾问审查或外部律师出具的法律意见。
- 涉及重大合同、国有资产交易、境外投资等，应依法履行正式决策及审批程序。
- 使用前请确认合同文本、招标文件、决策文件等材料的完整性与真实性。
